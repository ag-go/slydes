# Slydes

A presentation app using Fyne and Markdown

![](img/screenshot.png)

## Format

Each presentation is a simple markdown file.
It also supports toml front matter for configuration, such as theme.

```md
+++
theme = "matrix"
+++

# Heading
## Subheading

---

# Slide 2

Content for slide

---

# Bullets

* First
* Second
* Third

```
